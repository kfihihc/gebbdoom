\par
当我们上一部大热作品《WOLFENSTEIN 3D》发布时，公众几乎立刻涌现出大量自制工具：地图编辑器、声音编辑器、训练器等等。这一切都没有得到 id Software 关于文件格式或游戏结构的任何帮助。《DOOM》将作为一款开放式游戏发布。我们会为任何需要的人提供文件格式与技术说明。人们将能够轻松编写并分享从自制地图编辑器到通信与网络驱动等各种内容。}{1993 年初新闻稿}

引用：

DOOM 是一款无可争议的伟大游戏。它成功与持久的一个重要因素是大量用户自制的扩展内容。id Software 通过加入 -FILE 参数并采用直观易懂的数据格式，默认鼓励了这些作品。DOOM 基本由两个文件组成：DOOM.EXE 与 DOOM.WAD。DOOM.EXE 是“引擎”，负责显示与控制游戏；DOOM.WAD 包含引擎使用的全部图形、声音与地图/关卡数据。-FILE 参数允许将外部小型或大型“WAD”文件合并进来，从而修改任意数量的图形、声音与地图。
