\section{ShadowCaster}
\label{label_shadowcaster}
由 Raven 基于从 id Software 授权的技术开发。John Carmack 在 id Software 完成 Wolfenstein 3D 之后的技术研究中编写了 ShadowCaster 的 3D 引擎。该引擎具有衰减光照、地板与天花板贴图、可变高度墙体以及倾斜地板等特性。地图仍由具有正交墙体的方块构成。该引擎“速度约为 Wolfenstein 的一半”，但更适合 ShadowCaster 的探索玩法，而非 Doom 的快速动作节奏。
